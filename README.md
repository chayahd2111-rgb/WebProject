# Semantic News Navigator
Prompt Statement:
Topological, semantic news navigator
Tracks: AI powered search and information retrieval, the mission: engineer search engine that trancends keywords to map the conceptual distance of life data. You must project high dimensional embedding embedding onto a 2-D interactive canvas using PCA maths to visually cluster emerging news topics.

## 🧠 Overview
ThoughtGalaxy is an AI-inspired semantic search and visualization system that goes beyond keyword matching. It transforms textual data into an interactive 2D knowledge map where concepts are visually clustered based on their semantic relationships.

The system allows users to explore ideas in a spatial form, understand relationships between topics, and discover trending concepts through an interactive interface.

---

## 🚀 Features

- 🔍 Semantic-style search (topic-based exploration)
- 🌌 Galaxy View for concept clustering
- 🌍 Map View for trending topic visualization
- 🔗 Relationship explanation between connected ideas
- 📊 Conceptual distance-based positioning (PCA-inspired simulation)
- 🎯 Interactive nodes with click-based insights
- 🔥 Trending topics dashboard

---

## 🧠 Core Idea

Instead of traditional keyword search, ThoughtGalaxy:
- Represents text as conceptual nodes
- Maps relationships based on meaning
- Visualizes data in a 2D embedding space
- Simulates dimensionality reduction (PCA-style projection)

## 📊 System Design

1. User inputs a topic or selects trending data
2. System extracts related semantic content
3. Data is mapped into a 2D coordinate space
4. Nodes are rendered as an interactive graph
5. Connections show conceptual relationships
